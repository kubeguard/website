## guard login

Kubectl credential plugin

### Synopsis

Kubectl credential plugin. Visit here for more info: https://kubernetes.io/docs/reference/access-authn-authz/authentication/#client-go-credential-plugins

```
guard login [flags]
```

### Options

```

  -h, --help              help for login
```
